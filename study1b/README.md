# Study 1b — PEWS Replication with Permutation Entropy

**Pre-registration:** https://doi.org/10.17605/OSF.IO/EYMSF  
**Dataset:**          Sleep-EDF Expanded (PhysioNet) — same N=20 as Study 1  
**Status:**           Complete

---

## What this study tests

Study 1b replicates the PEWS prediction from Study 1 using **permutation
entropy (PE)** as the I_Sim proxy, with an independent analysis pipeline
applied to the same Sleep-EDF Expanded dataset.

PE **decreases** from Wake to N3 (Wake > REM > N2 > N3) — the opposite
direction from aperiodic slope. Using an independent measure with the
opposite direction tests whether the PEWS signal is a genuine pre-cognitive
early warning and not an artefact of the choice of complexity measure.

---

## Pre-registered hypotheses

| Hypothesis | Prediction | Result |
|---|---|---|
| **H1** | Lag > 0: PE variance peaks before mean crosses N3 threshold | Confirmed — 19/19, p < .001, r = .883 |
| **H2** | r ≥ .633 (replicates Study 1 effect size threshold) | Confirmed — r = .883 ≥ .633 |
| **H3** | β ≈ 0.546 (power-law variance exponent, exploratory) | Not confirmed — β = −0.462, CI [−0.774, −0.149] |

## How to run

Study 1b uses the complexity CSVs generated by Study 1's Step 2.
Run Study 1 Steps 0–2 first to download data and compute complexity.

```bash
# From the repo root:
python 00_download.py             # download Sleep-EDF (once)
python 01_preprocess.py           # extract sleep stages (once)
python 02_compute_complexity.py   # compute PE + aperiodic slope (once)

# Then run Study 1b from this folder:
cd study1b
python 01_pews_analysis.py        # pre-registered PEWS test with PE
python 02_figures.py              # publication figures
python 03_write_results.py        # auto-generate results text
```

## Analysis details

| Parameter | Value |
|---|---|
| Complexity measure | Permutation entropy (order=6, delay=1) |
| Direction | Decreases Wake→N3 (Wake > N3) |
| N3 threshold | 75th percentile of per-subject N3 PE epochs |
| Crossing | `roll_mean <= threshold` (PE falls into N3) |
| Rolling window | 6 epochs = 3 minutes (centred) |
| Pre-transition window | 20 epochs = 10 minutes |
| Test | One-sided Wilcoxon signed-rank, α = .05 |
